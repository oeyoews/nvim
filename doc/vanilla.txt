TIPS *vanilla-tips*

	      - For better edit your vanilla.txt, first use <cmd> set filetype=help <cr>,
	        to get help syntax highlight

	      - Generate ASCII use :r !figlet <txt>

	      - 'go'          jump beginning of documentation

	      - <C-]> or K to jump current tags, like keyword

	      - <C-o> in insert mode, can enter normal mode temporary

	      - 'rtp'  nvim -c "set+=./" [ file ]

	      - 'stdpath'

	      - 'center [nu]' align text. use :formattin to see more info

	      - z[hjkl][H L] when set nowrap, to scroll

	      - 'gM' move cursor to middle of line

	      - 'gq' merge multi lines to one line in visual mode, like 'J'
